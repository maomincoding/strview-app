# strviewCLI
![strview.js](https://www.maomin.club/data/strview/logo.png)

Project framework based on strview.js.

## About

You can click on the link below to learn about `strview.js`.
```
https://github.com/maomincoding/strview.git
```

## Install

```bash
npm install
```
or
```bash
yarn install
```

## Start up

```bash
npm run start
```
or
```bash
yarn start
```

## Build

```bash
npm run build
```
or
```bash
yarn build
```
